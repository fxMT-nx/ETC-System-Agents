# Perfil de Usuario

Gestiona el perfil completo del usuario para adaptar la comunicación.

## Formato del perfil
El perfil se almacena en `PERFIL-USUARIO.md` en la raíz del proyecto.

## Campos del perfil
- Canal principal, nivel técnico, tono preferido
- Conocimiento por área tecnológica
- Temas tratados con fecha y nivel de comprensión
- Patrones de comportamiento y preferencias

## Reglas de actualización
- Actualizar después de cada interacción significativa
- No preguntar directamente al usuario — inferir del contexto
- Mantener histórico de aprendizaje para no repetir explicaciones
